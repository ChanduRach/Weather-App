This project includes a source code of weather indicator web page completly done by JavaScript 

Semantic ui is used a css frame work for a better look and reusable code

Some basics of JavaScript functions are used like JavaScript Objects, functions, If statements, ... etc

API used for getting the weather info - http://api.openweathermap.org/data/2.5/weather

When the user enters the city name and clicks on the search button the city name,  temperature value (celcius), humidity will be displayed in the UI

The API provider is http://www.OpenWeatherMap.org
